# Transient plugin for Craft CMS 3.x

Store variables in Twig code which will be available to other templates for the duration of the page load.

## Requirements

This plugin requires Craft CMS 3.0.0 or later.

## Installation

To install the plugin, follow these instructions.

1.  Open your terminal and go to your Craft project:

        cd /path/to/project

2.  Then tell Composer to load the plugin:

        composer require elivz/transient

3.  In the Control Panel, go to Settings → Plugins and click the “Install” button for Transient.
